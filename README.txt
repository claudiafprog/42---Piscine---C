# 42 Piscine - C & Shell

This repository contains the exercises completed during the 42 Piscine (C programming bootcamp).

## Overview

The Piscine is an intensive selection and learning process at 42, focused on building strong foundations in programming, problem-solving, and autonomy.

This repository includes exercises in:
- C programming
- Shell basics

## Structure

### C Projects
- C00 → Introduction to C fundamentals
- C01 → Basic functions and logic
- C02 → Strings and basic data manipulation
- C03 → Deeper string manipulation and logic
- C04 → Integer/string handling and conversions
- C05 → Recursion and more advanced logic
- C06 → Arrays and arguments handling
- C07 → Memory allocation and dynamic concepts
- C08 → Advanced C topics and structured programming

### Shell Projects
- Shell00 → Basic shell commands and navigation

### Rush Projects
- Rush00 → Group-based coding challenge focused on logic, collaboration, and time management

## Learning Approach

This repository reflects a progressive learning process focused on:
- writing clean and structured C code
- improving problem-solving skills
- working under time constraints
- building autonomy in debugging

## Notes

The code is not optimized for production but represents learning progression during the 42 Piscine selection process.
